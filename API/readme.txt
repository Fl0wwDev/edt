Exemple de requête sur l'API : 
http://localhost:1337/edt?group=s4info&iut=mmi

La requête prend deux paramètres : 
group et iut. ils sont obligatoires pour que la requête fonctionne.
Pour voir les valeurs que peuvent prendre les paramètres, vous pouvez lire le fichier request.json
